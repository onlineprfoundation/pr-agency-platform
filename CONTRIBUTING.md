# Contributing to Online PR

Thank you for your interest in contributing.

## How to contribute

1. **Fork** the repository
2. **Create a branch**: `git checkout -b feature/your-feature` or `fix/your-fix`
3. **Make changes** following the code style below
4. **Commit** with clear messages
5. **Open a Pull Request** against the main branch

## Code style

- **PHP**: PSR-12, Laravel conventions
- **Blade**: Indent with 4 spaces
- **JavaScript**: ES6+, consistent with existing code

## Pull request process

- Ensure tests pass (if applicable)
- Update CHANGELOG.md for notable changes
- Keep PRs focused; split large changes into smaller PRs

## Questions

Open an issue for questions or discussions.
